# Fake Identity

Plausible fake identity generator (in Romanian).

![Screenshot of identitate falsa (fake identity) generator.](screenshot.png)

## Usage

Clone this repo. Build and run it by typing:

    npm run up

Go to [localhost:3000](http://localhost:3000) to see it. See other [supported
commands][sc].
